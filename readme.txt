## Setup
Clone the repo:
```
git clone https://github.com/TTTANGCUHK/WeatheringWithMe.git
```

Change directory:
```
cd WeatheringWithMe
```

Install dependecies:
```
npm install
```

## Running dev environment
```
npm run dev
```
The server will be started at http://localhost:3000
